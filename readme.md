Для настройки и запуска проекта:

1. Перейдите в директорию, где вы хотите создать проект, и клонируйте репозиторий:

git clone https://github.com/humam11/test

2. Перейдите в директорию проекта:

cd test

3. В директории проекта установите необходимые зависимости:

npm install

4. Запустите сервер разработки:

npm run dev
